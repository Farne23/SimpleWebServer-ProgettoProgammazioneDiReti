Web Server Statico in Python

Anno Accademico 2024/2025

Benvenuto!

Questo progetto dimostra la realizzazione di un semplice web server in Python capace di gestire richieste HTTP GET e servire file statici come HTML, CSS e immagini.

Il server è multithread, supporta il logging delle richieste, la gestione degli errori 404 e rispetta i MIME types. Tutto il contenuto è responsive per dispositivi mobili.
