Simple Web Server

Realizzato da Michele Farneti: MATRICOLA 0001080677

Web service semplice realizzando mediante l'ausilio di python, in grado di gestire contemporaneamente richieste da plurimi client. Durante la navigazione delle pagine sarà possibile assistere a messaggi personalizzati riportanti le richieste HTTP eseguite sulla conosole da cui è stato avviato il server, riferite ad ogni client connesso.

La chiusura del server può essere effetuata in maniera sicura in qualsiasi momento tramite la combinazione di tasti Ctrl+C effetuata da linea di comando.
